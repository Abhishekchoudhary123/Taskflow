# ⚡ Quick Start Guide

## 🎯 Push to GitHub (3 Simple Steps)

### Step 1: Create Repository on GitHub
1. Go to: https://github.com/new
2. Repository name: `taskflow`
3. Click "Create repository" (don't initialize with anything)

### Step 2: Add Remote & Push
```bash
cd task-manager
git remote add origin https://github.com/Abhishekchoudhary123/taskflow.git
git push -u origin main
```

### Step 3: Authenticate
- Username: `Abhishekchoudhary123`
- Password: Use Personal Access Token from https://github.com/settings/tokens

## 🚀 Run Locally

### Backend
```bash
cd task-manager/backend
npm install
npm run dev
```
Server runs on: http://localhost:5000

### Frontend
```bash
cd task-manager/frontend
npm install
npm run dev
```
App runs on: http://localhost:5173
